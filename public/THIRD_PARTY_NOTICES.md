# Third-Party Notices and Content Provenance

## Course content and assets

The lesson explanations, example sentences, reading passages, listening scripts, questions, and
answers in this repository were authored specifically for this project. The favicon is an original,
code-authored SVG made from basic geometric and text elements.

Listening exercises use the learner's browser-provided Web Speech API to read the authored French
scripts at runtime. This repository does not contain or redistribute recorded voices, speech models,
music, photographs, video, or stock artwork.

Translations requested by a learner are fetched at runtime and cached only in that learner's browser;
translated results are not included in the repository or production bundle.

## Web fonts

The site requests these families from Google Fonts. They are not stored in this repository. Each is
licensed under the [SIL Open Font License 1.1](https://openfontlicense.org/):

- [Fraunces](https://github.com/google/fonts/tree/main/ofl/fraunces)
- [Public Sans](https://github.com/google/fonts/tree/main/ofl/publicsans)
- [JetBrains Mono](https://github.com/google/fonts/tree/main/ofl/jetbrainsmono)
